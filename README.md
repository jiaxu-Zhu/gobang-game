# 五子棋游戏

一个简洁美观的在线五子棋游戏，支持双人对战、悔棋和重新开始功能。

## 功能特点

- 🎮 15×15标准棋盘
- ⚫⚪ 黑白双人对战
- 🔄 悔棋功能
- 🏆 自动检测获胜五子并高亮显示
- 📱 响应式设计，支持移动端
- 🎨 精美的渐变UI设计

## 快速开始

1. 将此仓库上传到GitHub
2. 在GitHub仓库设置中启用GitHub Pages
3. 访问 `https://你的用户名.github.io/仓库名/` 即可开始游戏

## 本地运行

直接在浏览器中打开 `index.html` 文件即可。

## 游戏规则

- 黑棋先行
- 双方轮流在棋盘交叉点落子
- 率先形成横向、纵向或对角线五子连珠者获胜
- 棋盘下满无五子连珠则为平局
